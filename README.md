## Utility 1 housing cost calculator 
lightweight feature heavy calculator for comparing houses because im looking at buying a house
## Features
- create and save houses in a local json file which can be used between devices if transfered
- includes amortization
- supports fields for interest rate, home insurance, private mortgage insurance, utilities, extra principal payments, down payment etc.
- includes minimal closing cost estimate
- Uses only packages in the python standard library

## Utility 2 student loan calculator
A Tkinter app for planning and tracking student loan repayment with both single-loan and portfolio-level insights.

### Features
- Create and save loans with source/servicer, loan name, type, balance, rate, and repayment settings.
- Supports weekly planning inputs such as autopay discount, grace period behavior, and extra monthly payments.
- Shows effective rate, minimum payment, planned payment, total interest, total repayment cost, and months to payoff.
- Includes an amortization tab with payment schedule and CSV export.
- Includes a portfolio summary tab with totals by source and a visual chart of projected repayment.
- Saves all loan records to `Student_loans_data.json` and reloads automatically on launch.
- Uses only packages in the python standard library


## Utility 3 budget planner
A Tkinter budgeting app for monthly and yearly planning with flexible income/savings frequencies and unlimited custom expenses.

### Features
- Enter income and savings with frequency options: Weekly, Biweekly, Monthly, or Annually.
- Add any number of named expenses and choose payment frequency per expense.
- Automatically converts all entries into monthly and yearly totals.
- Displays monthly and yearly income, savings, expenses, and net amounts.
- Save multiple named budgets and reload them into the planner.
- Data is saved to `budgeting_data.json` and loaded automatically on app start.
- Uses only packages in the python standard library
